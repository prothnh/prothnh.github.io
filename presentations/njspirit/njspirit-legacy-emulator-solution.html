🔗 REST API Endpoints

Comprehensive RESTful API exposing legacy functionality

👥 Person Management API

GET /rest/persons - List all persons
GET /rest/persons/{id} - Get person by ID
POST /rest/persons - Create new person
PUT /rest/persons/{id} - Update person
DELETE /rest/persons/{id} - Delete person

📁 Case Management API

GET /rest/cases - List all cases
GET /rest/cases/{id} - Get case by ID
POST /rest/cases - Create new case
PUT /rest/cases/{id} - Update case
DELETE /rest/cases/{id} - Delete case

🏥 Health & Monitoring API

GET /rest/health - Basic health status
GET /rest/health/live - Kubernetes liveness probe
GET /rest/health/ready - Kubernetes readiness probe
GET /rest/health/detailed - Comprehensive system info
GET /rest/health/database - Database connectivity
GET /rest/openapi.json - OpenAPI specification
