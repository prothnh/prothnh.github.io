🚀 Deployment Options

🏠 Local Development

  1. Configure Oracle Database
  2. Update database.properties
  3. Run: mvn jetty:run
  4. Access: http://localhost:8080

🐳 Docker Deployment

  1. Build: docker build -t njspirit-legacy .
  2. Run: docker-compose up -d
  3. Access: http://localhost:30082
  4. Monitor: docker logs -f

☸️ Kubernetes Production

  1. Create namespace: kubectl create ns njspirit
  2. Apply secrets: kubectl apply -f secrets.yaml
  3. Deploy: kubectl apply -f k8s-deployment.yaml
  4. Verify: kubectl get pods -n njspirit
